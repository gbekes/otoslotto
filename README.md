# Csal-e az ötöslottó sorsolás? 

Ez egy kis repo, amelyben minden 100% claude code készített, teljes munka 20p volt. 

Tud-e az AI cikket írni? Ötös lottó rekord nyeremény elé gondoltam megnézem. Mindenki el tudja dönteni. 

## Összesen három kör volt. 

```
"Készíts egy új mappát otoslotto neven. Egy kis cikket szeretnek irni, arrol hogy csalnak-e a magyar ötöslottó húzásnál, vagyis vannak-e gyanusan sokszor húzott számok. Gyújts adatokat a szerencsejatek oldalarol. keszits egy adat file-t. Majd tervezd meg a cikket. Utána irj kódot, amely legyártja a szükséges ábrákat és teszteket. ird meg a cikket magyarul elég összesen 4-500 szó. Mentsd el a cikk szöveget txt-ben, és külön legyenek az ábrák, táblák számolások. Nem kell semmi más. Nyugodtan haladj, az otoslotto almappaban barmit csinalhats, python is futhat kedvedre."
```

Megnéztem és kicsit uncsi volt. Ezért. 

```
Van bármilyen más elemzési módszer amely hasznos lenne? Ha van csináld meg és egészítsük ki a cuccot. 
```

Mondott egy csomó ötletet. 

```
Okés. csinálj belőle egy cikket, lehet 800-1000 szavas, egy 444, telex stílusában. Olvasmányos de pontos (nem kell a tesztek neve, de monjduk el mit nézünk. Legyen 3-4 ábra, 2-3 ami a legfontosabb + 1 érdekes (5 -- 8 abra egyik). Legyen egy html, amit fel tudok rakni a honlapomra. GitHub\gaborbekes-site
```

## Mit csináltam én

Alapötlet. És a promptok. Nem szerkesztettem a szöveget vagy ábrákat. 

Nem lett tökéletes, de nem rossz.
